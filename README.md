# dog_ear

Windows desktop sticky notes with animal outlines.

## Web preview

Open `index.html` in a browser.

## Windows desktop app

Install dependencies:

```sh
npm install
```

Run the desktop app:

```sh
npm start
```

Build a Windows app:

```sh
npm run build:win
```

The Electron version opens a transparent full-screen overlay. Notes and the `+`
button are clickable, while empty transparent areas pass mouse clicks through to
the desktop or apps behind it.

Quit shortcut:

```text
Ctrl + Shift + Q
```
